NAME

svm_model - manual page for svm_model of 0.01  

SYNOPSIS

svm_model [options] model-file  

DESCRIPTION

TinySVM - tiny SVM package Copyright (C) 2000 Taku Kudoh All rights reserved.  

options:

-n, --sv-num
show number of Support Vectors.
-m, --margin
show estimated margin.
-d, --vc-dimension show estimated VC dimension.
-v, --version
show the version of TinySVM and exit.
-h, --help
show this help and exit.
